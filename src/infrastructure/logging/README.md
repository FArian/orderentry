# 📈 Logging

Server-side logging utilities.

## ⚙️ Rules

- Only used in API routes and server components
- Never import in `"use client"` files
- Log level controlled via `LOG_LEVEL` env var (see `EnvConfig`)
- File logging controlled via `LOG_FILE` env var; disabled by default
